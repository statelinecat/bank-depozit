# bank-depozit

## Расчет дохода по банковскому депозиту

Пользователь вводит:
- сумму вклада
- срок вклада
- процент вознаграждения по вкладу

Программа вычисляет общую сумму вклада, 
которая будет на счете в конце срока вклада.
